Создание API-ключей и выбор группы

1
Перейдите в «Управление токенами»

В боковом меню консоли выберите «Управление токенами». По умолчанию у вас уже создан первый токен (API-ключ).

Для удобного переключения между группами вы можете создавать сколько угодно токенов.

Пункт «Управление токенами» в боковом меню
2
Нажмите «Редактировать»

Рядом с нужным токеном нажмите «Редактировать».

Кнопка «Редактировать» напротив токена
3
Выберите группу токенов

Можно дать название каждому токену для удобства. В поле «Группа токенов» выберите подходящую группу.

⚠️ Без выбранной группы ключ работать не будет — платформа выдаст ошибку.

Подключение SillyTavern

Подключение напрямую

Подключение к моделям в SillyTavern осуществляется напрямую через Chat Completion. Заполните следующие поля:

  • API — Chat Completion
  • Источник для Chat Completion — Claude (или Google AI Studio для Gemini)
  • Раскройте блок «Прокси»
  • Адрес прокси-сервера — https://api.linkapi.ai/v1
  • Для Gemini — без v1
  • Дополнительные эдпоинты указаны в дискорде
  • Пароль от прокси — ваш API-ключ
Настройки API — Chat Completion, источник Claude, блок «Прокси» с адресом и ключом

Выбор модели

Модели подтягиваются автоматически

Выбирать и копировать модели с сайта не нужно. После ввода ключа автоматически подтянутся модели выбранной группы.

В выпадающем списке версий выберите нужную модель.

⚠️ Если в списке есть варианты с цифрами в названии и без — выбирайте с цифрами (кроме claude-opus-4-6).

Группа Once — через Custom Models

Для группы Once модели берутся с сайта

Только для группы Once модели нужно добавлять вручную — они не подтягиваются автоматически.

Для этого используется расширение Custom Models для SillyTavern:

👉 github.com/LenAnderson/SillyTavern-CustomModels

1
Скопируйте название модели с площадки

Откройте «Площадка моделей» на сайте LinkAPI. Выберите группу Once, найдите нужную модель и скопируйте её название по иконке копирования.

Площадка моделей — фильтр по группе Once
2
Добавьте модель через расширение

Установите расширение и в поле Custom Models вставьте скопированное название модели.

💡 Если расширение не работает — попробуйте сменить язык таверны на английский.
